Overview

What?

HTML in Hyde is a HTML Freewrite - [an event] in which people get together and write HTML documents. events - html energy lists similar events all around the world, like the Ontario based HTML in the Park!

Schedule

  • 12:00 - setup
  • 12:15 - call to action
  • 12:22 - code code code
  • 2:10 - demos
  • 2:30 - keep cooking (if you want)

Why?

Ask a mid-20s software engineer "why did you start coding?" - a non-trivial amount will say "I was trying to make my Tumblr look cool".
If they're a bit older, they might say Myspace. Older still, Geocities.
I feel like that there's a prevailing nostalgia for an earlier internet, where people had their own little space that they could express themselves creatively.

New developments

Fortunately, there's a growing community of people who share this nostalgia.
Diagram Website - An internet map has attempted to paint a picture of this community (which is, unsurprisingly, quite decentralized). Its broad categorizations are:
  • Fun - indie games, silly useless software, whimsical web…
  • HTML - net.art, handmade web, folder poetry…
  • Publish - sheet sites, internet radio, folk databases…
  • Atmosphere - lightweight web, tiny internets, slow web…
  • Ecology - permacomputer, feral web, low tech…
  • Multiplayer - live streaming, webrings, social networks…
  • Text - text only websites, ascii art, no css club…
  • Myth - cyber witches, ritual technology, cccentric engineering…
  • Independence - indie web, feminist servers, view source…
  • Idiosyncratic - web shrines, digital gardens, home cooked apps…
  • Small - smolnet, organic internet, spartan web…
^ this is only a subset of Diagram Website, so have a dig around in the full map. It's a great starting point to search for inspiration for something to build at a HTML Freewrite.

Resources

For complete beginners

  • Getting started with the web - MDN
    This will guide you through installing basic software, HTML basics, CSS basics, and hosting.
  • Neocities
    Geocities reincarnated. Very basic web hosting. Recommended hosting option.
  • Glitch
    A more "modern" browser-based service for building a website with HTML. Might be better for someone who has had some light exposure to web development and finds Neocities too basic.
  • mmm.page
    While I personally really like mmm.page, it's not really in the spirit of the event to use it. But it's great for building a cool site really quick. So... use this if you need to leave early.

For everyone else

  • GitHub Pages
    GitHub Pages lets you host static websites directly from a canonical username.github.io repo, or any other project repo. For example, my MaxwellBo/maxwellbo.github.io repo maps to maxbo.me, while MaxwellBo/cursors maps to maxbo.me/cursors. Recommended.
  • Val Town
    If GitHub Gists could run. And AWS Lambda was fun. Val Town is a social website to write and deploy TypeScript. Build APIs and schedule functions from your browser.
    I used Val Town power the RSVP form, with the code hosted here: val.town/v/mbo/htmlinhydeRSVPs. You can even deliver HTML from a val! Super cool!
